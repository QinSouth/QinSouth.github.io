iReport 是为JasperReports Library和JasperReports Server设计的报表可视化设计器。
介绍内容采自百度百科-ireport

iReport 的最新版本是 iReport 5.6.0，但也是在2014年5月份发布的，之后再也没有更新过，而且最高只支持到 JDK 1.7（已经被淘汰了）。如果你的应用场景是为了做图表展示的话，建议使用其它软件。

一、准备文件

iReport-4.0.0 百度网盘下载地址：iReport-4.0.0_Package，提取码：3e2c，选择其中一个下载即可，zip直接解压，exe则进行安装，效果完全一样

iReport-4.0.0 只支持到 JDK 1.7，但可以不配置到环境变量。不影响电脑上原来的JDK环境。
JDK 1.7 百度网盘下载地址：Java SE Development Kit 下载，提取码：qjzp
